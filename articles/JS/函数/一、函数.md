## 1. 函数参数求值

### (1) 传值调用

进入函数体之前，先计算参数值，在将值传入函数

**① 优点**：比较简单

**② 缺陷**：对参数求值时，实际上还没用到这个参数，可能造成性能损失

```javascript
function f(x){
  return x * 2;
}
var a = 1;
f(a + 4); //等价于 f(5)
```

### (2) 传名调用

直接将参数表达式传入函数，用到的时候再求值

```javascript
function f(x){
  return x * 2;
}
var a = 1;
f(a + 4); //等价于 (a + 4) * 2
```
