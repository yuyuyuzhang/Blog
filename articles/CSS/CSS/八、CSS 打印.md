# 八、CSS 打印

## 1. 媒体查询打印

```css
@media print {
    /* 控制打印样式 */
}
```

## 2. 打印相关样式

```js
page-break-after  //设置指定元素之后插入分页符(auto:允许元素后插入分页符,always:必须元素后插入分页符,avoid:禁止元素后插入分页符,left:元素后插入足够的分页符直到一张空白的左页为止,right:元素后插入足够的分页符直到一张空白的右页为止)
page-break-before //设置指定元素之前插入分页符(auto:允许元素前插入分页符,always:必须元素前插入分页符,avoid:禁止元素前插入分页符,left:元素前插入足够的分页符直到一张空白的左页为止,right:元素前插入足够的分页符直到一张空白的右页为止)
page-break-inside //设置指定元素内部插入分页符(auto:允许元素内部插入分页符,avoid:禁止元素内部插入分页符)
```
